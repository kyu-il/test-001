# Contributing Guide

프로젝트 기여 시 `CONTRIBUTING.md`의 규칙을 자동으로 적용합니다.

## 자동 적용 규칙

1. **브랜치**: `feat/`, `fix/`, `docs/`, `refactor/`, `test/` 접두사 사용
2. **커밋**: `feat:`, `fix:`, `docs:`, `refactor:`, `test:` 형식
3. **PR**: 변경 사항 요약 필수
4. **테스트**: PR 전 `npm test` 통과 필수

## 작업 시 확인

- `CONTRIBUTING.md` 를 읽고 규칙을 따르세요
- 커밋 메시지가 규칙에 맞는지 확인하세요
- PR 생성 시 변경 사항을 요약하세요
